## LifestyleProps[​](lifestyle.html#lifestyleprops)
参数类型必填说明publicId`string`否必填，生活号 ID（即生活号 APPID），必须是当前小程序同账号主体且已关联的生活号。memo`string`否文案。支持商家自定义，最多展示一行。onFollow`CommonEventFunction`否当用户关注生活号成功后执行。
